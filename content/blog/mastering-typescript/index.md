# Mastering TypeScript

TypeScript is a powerful superset of JavaScript that adds static typing and other features.

## Key Features

1. Static typing
2. Type inference
3. Interfaces
4. Decorators
5. Enhanced IDE support

## Getting Started

First, install TypeScript globally:

```bash
npm install -g typescript
```
